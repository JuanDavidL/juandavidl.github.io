Sobre mí

Soy Gameplay Developer especializado en la creación de mecánicas interactivas y sistemas escalables con Unity y C#. Mi enfoque principal es el desarrollo de sistemas de juego modulares, utilizando herramientas como ScriptableObjects para optimizar el flujo de trabajo y la flexibilidad del diseño.

Cuento con una formación sólida como Ingeniero de Software, lo que me permite aplicar principios de Clean Code, patrones de diseño y una gestión profesional de versiones con Git en cada proyecto.
